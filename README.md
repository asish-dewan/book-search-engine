
# Book Search Engine

Google Books API search engine built with a RESTful API refactored to be a GraphQL API built with Apollo Server.

## Installation

Install my-project with npm

```bash
$ git clone https://github.com/asish-dewan/book-search-engine
$ cd project
$ npm install
```

### Development mode

```bash
$ npm run develop
```

### Build app
```bash
$ npm run build
```
## Technologies used

 - MongoDB
 - Express
 - React
 - Node.js
 - Apollo Server
 - Graph QL
